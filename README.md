# Shopee Web API PHP

This is a PHP wrapper for [Shopee's Web API](https://open.shopee.com/developer-guide/12)


## Requirements
* PHP 8.2 or later.

## Installation
Install it using [Composer](https://getcomposer.org/):

```sh
composer require ryanvelbon/shopee-web-api-php
```
